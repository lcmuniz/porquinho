# Technical Preferences

## User Technical Stack Preferences

**Frontend:**
- Framework: Vue 3.5+
- Meta-framework: Não aplicável (SPA puro)
- Type Safety: TypeScript
- Styling: Tailwind CSS
- Component Library: shadcn-vue (copy-paste approach)

**Backend:**
- Language: Java 21 LTS
- Framework: Spring Boot 3.4.x / 4.0.3
- Build Tool: Maven
- ORM: Spring Data JPA + Hibernate

**Database:**
- Primary: PostgreSQL 16+
- Hosting: Supabase (managed PostgreSQL + Auth)
- Rationale: OAuth Google built-in, Row Level Security

**Deployment:**
- Platform: VPS Hostinger
- Orchestration: Dokploy
- Strategy: Self-hosted containers

**External Services:**
- Payment: Stripe
- Authentication: Google OAuth (via Supabase Auth)
- IA/ML: Hosted API (OpenAI ou Claude)
- Analytics/Monitoring: A definir

**Experience Level:**
- Java: Experiência sólida
- Vue: Experiência sólida
- Stack familiar: Preferência por tecnologias conhecidas
