## Desafios e Aprendizados

### 1. Versoes desatualizadas na documentacao de arquitetura

**Ocorreu em:** Stories 0.1, 0.2, 0.4, 0.6

| Componente | Planejado | Real |
|------------|-----------|------|
| Vite | 6.x | 7.3.1 |
| Spring Boot | 3.4.x | 3.5.0 (3.4.x indisponivel no Initializr) |
| Docker image backend | openjdk:21-jdk-slim | eclipse-temurin:21-jre-alpine (openjdk descontinuado) |
| Tailwind CSS | v3 (tailwind.config.js) | v4 CSS-first (abordagem completamente diferente) |

**Impacto:** Cada discrepancia gerou trabalho extra de descoberta e atualizacao de docs mid-story.

**Acao:** Verificar versoes reais disponiveis (npm, Maven Initializr, Docker Hub) antes de criar stories para cada epic.

O projeto tem git inicializado localmente mas nao foi publicado no GitHub. Os workflows de GitHub Actions existem no codigo mas nunca foram testados.

**Acao:** Criar repositorio GitHub e testar CI/CD — item critico antes do Epic 1.
